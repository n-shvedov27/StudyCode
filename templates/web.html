Создание сайтов

Курс будет интересен всем, кто хочет научиться создавать свои сайты с нуля. Ребята будут писать код в профессиональной среде разработки (WebStorm), поймут, как устроены сайты, получат необходимые навыки для самостоятельной работы.

Курс проходит в два этапа.

На первом этапе дети будут изучать язык гипертекстовой разметки (HTML), каскадные таблицы стилей (CSS) и начнут изучать самый востребованный язык программирования JavaScript.

На втором этапе ученики продолжат более углубленно изучать JavaScript в связке с фреймворком jQuery, научатся создавать сервер для своего сайта, работать с базой данных, а так же узнают как разместить сайт в сети.

Задачи курса:
- Познакомить с HTML, CSS и JavaScript
- Помочь написать первые программы
- Дать базовые навыки разработки сайтов
- Обучить основам веб-дизайна
- Научить применять фреймворки при разработке
